What Are Some of the Most Common Ingredients Used in Italian Recipes?

Purpose:

Display the relationship between different ingredients and recipes in
order to see which ingredients you should always be stocked up on.

  • Source

A Basic Interactive Network

The data used for this was taken from this list of authentic Italian pastas and manually entered into an excel spreadsheet. Further data transformations were made within R in order create the edge and node lists. These data transformations can be viewed in the source code link above.

Sankey Network Visualization

From the Sankey network we can that some of the most needed ingredients that we need in our kitchen are olive oil, garlic and tomatoes. With the given data, chilis are needed pretty often as well surprisingly, but they typically aren't mandatory to use in Aglio e Olio.

Colored Interactive Network

Click a recipe and drag it to see the ingredients it requires. Conversely, click and drag an ingredient to see how many recipes it is needed in.
